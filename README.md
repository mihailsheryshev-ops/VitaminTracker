# VitaminTracker — Трекер витаминов и БАДов

Мобильное приложение для Android — отслеживай приём витаминов и БАДов, следи за нормами и избытком.

## Описание
Добавляй витамины с указанием дозы и суточной нормы. Нажимай «Принял» когда принял таблетку — приложение считает сколько мг принято за сегодня и показывает прогресс. Данные сбрасываются автоматически каждый день.

## Функции
- Добавление витамина с названием, дозой, нормой, описанием и симптомами избытка
- Кнопка «Принял» — засчитывает дозу за сегодня
- Прогресс-бар показывает сколько принято от нормы
- Статус: не принят / частично / норма / превышение
- Автоматический сброс счётчика каждый день
- Удаление витаминов из списка

## Требования
- Android 7.0 и выше (API 24+)

## Запуск
1. Скачай и установи Android Studio
2. Открой проект: File → Open → выбери папку проекта
3. Подключи Android-устройство по USB
4. Нажми Run ▶

## Стек технологий
- Kotlin
- Android SDK
- SharedPreferences — локальное хранение данных
- Gson — сериализация объектов
